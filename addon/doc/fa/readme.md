# دستیار حرفه‌ای بینایی

**دستیار حرفه‌ای بینایی** یک دستیار هوش مصنوعی پیشرفته و چندحالته برای NVDA است. این ابزار از مدل‌های Gemini گوگل استفاده می‌کند تا امکانات هوشمند خواندن صفحه، ترجمه، تایپ صوتی و تحلیل اسناد را فراهم کند.

_این افزونه به مناسبت روز جهانی افراد دارای معلولیت برای جامعه‌ی کاربران NVDA منتشر شده است._

## ۱. راه‌اندازی و پیکربندی

به **منوی NVDA > اولویت‌ها > تنظیمات > دستیار حرفه‌ای بینایی** بروید.

- **کلید API:** نیاز است. برای امنیت، این کادر بطور پیش‌فرض پوشیده شده. برای مشاهده‌ی این کادر از گزینه‌ی «نمایش کلید API» استفاده کنید. یک کلید رایگان از [Google AI Studio](https://aistudio.google.com/) بگیرید.
- **مدل:** از میان مدل‌های **Flash** (سریعترین/رایگان) یا **حرفه‌ای** (هوشمندی بالاتر)، یکی را بر اساس نیازهایتان انتخاب کنید.
- **نشانی پراکسی:** اختیاری. اگر گوگل در منطقه‌ی شما مسدود است، از این گزینه استفاده کنید. شما به یک نشانی سرور (URL) نیاز دارید که درخواست‌های شما را دریافت کرده و آنها را به API جمینای ارسال کند.
  > **توجه:** این برای پراکسی‌های استاندارد VPN یا SOCKS (مثل `127.0.0.1:1080`) **نیست**. باید یک نشانی وب باشد (مثلا `https://my-custom-proxy.com`) که به عنوان پلی به گوگل عمل کند.
- **زبان‌ها:** زبان‌های مبدأ، مقصد و زبان پاسخ‌های هوش مصنوعی را تعیین کنید.
- **جابجایی هوشمند:** در صورتی که متن مبدأ با زبان مقصد یکسان باشد، زبان‌ها را به‌طور خودکار جابجا می‌کند.
- **خروجی مستقیم:** پنجره‌ی گفتگو را نمایش نمی‌دهد و پاسخ را مستقیماً از طریق گفتار اعلام می‌کند.
- **ادغام در کلیپ‌بورد:** پاسخ هوش مصنوعی را بطور خودکار در کلیپ‌بورد کپی می‌کند.

## ۲. لایه‌ی فرمان‌ها و میانبرها

برای جلوگیری از تداخل صفحه‌کلیدی، این افزونه از یک **لایه‌ی فرمان** استفاده می‌کند.
۱. کلیدهای **NVDA + Shift + V** (کلید اصلی) را برای فعال‌سازی لایه‌ی یادشده فشار دهید. یک بوق خواهید شنید.
۲. کلیدها را رها کنید، سپس یکی از این تک‌کلیدها را فشار دهید:

| میانبر                    | عمل‌کرد             | توضیح                                                                 |
|-----------------------------|----------------------|-----------------------------------------------------------------------------|
| **T**           | مترجم هوشمند     | متن زیر مکان‌نمای پیمایشگر یا متن انتخاب‌شده را ترجمه می‌کند.     |
| **Shift + T**           | مترجم کلیپ‌بورد | محتوایی که در حال حاضر در کلیپ‌بورد است را ترجمه می‌کند.            |
| **R**           | پالایش‌کننده‌ی متن         | متن را خلاصه، اشکالات گرامری را برطرف، توضیح هوش مصنوعی راجع به متن را دریافت، یا **پرامپت‌های سفارشی** را اجرا می‌کند.                 |
| **V**           | مشاهده‌ی شیء        | پیمایشگر جاری را توصیف می‌کند.                         |
| **O**           | مشاهده‌ی تمام صفحه   | چیدمان و محتوای کل صفحه را تحلیل می‌کند.                                  |
| **Shift + V** | تحلیل آنلاین ویدئو    | ویدئوهای **یوتیوب**، **اینستاگرام** یا **توییتر (X)** را از طریق URL تحلیل می‌کند.                        |
| **D**           | تحلیل سند    | با فایل‌های PDF/TXT/MD/PY گفتگو کنید.                                              |
| **F**           | نویسه‌خوان نوری (OCR) فایل             | فایل‌های عکس،PDF و TIFF را مستقیما OCR کنید. از TIFF چندصفحه‌ای پشتیبانی می‌شود.                                            |
| **A**           | رونویسی فایل صوتی  | فایل‌های MP3/WAV/OGG را رونویسی کنید.                                               |
| **C**           | حل‌کننده‌ی کپچا       | کپچا را دریافت و بطور خودکار حل می‌کند.                                 |
| **S**           | تایپ صوتی هوشمند      | گفتار را به متن تبدیل می‌کند. برای شروع یک بار بزنید. برای توقف و تایپ کردن، دوباره بزنید.       |
| **L**           | اعلام وضعیت     | وضعیت فعلی افزونه را اعلام می‌کند. (مثلا، «در حال آپلود»، «بیکار»، و غیره.                |
| **U**           | بررسی بروزرسانی         | Github را برای دریافت آخرین نسخه از افزونه بررسی کنید.                                            |
| **H** | راهنمای فرمان‌ها | فهرستی جامع از تمام کلیدهای میانبر موجود و توضیحات آنها را در لایه‌ی فرمان‌ها نمایش می‌دهد. |

## ۳. پرامپت‌ها و متغیرهای سفارشی

فرمان‌ها را در تنظیمات، در گزینه‌ی `نام:متن پرامپت` ایجاد کنید. (فرمان‌ها را با نویسه‌ی «خط عمودی» `|` یا خط‌های جدید از هم جدا کنید).

### متغیرهای موجود

| متغیر         | توضیح                                      | نوع ورودی       |
|------------------|--------------------------------------------------|------------------|
| `[selection]`    | متن در حال حاضر انتخاب‌شده                          | متن             |
| `[clipboard]`    | محتوای کلیپ‌بورد                                | متن             |
| `[screen_obj]`   | عکس فوری از پیمایشگر                   | تصویر            |
| `[screen_full]`  | عکس فوری از تمام صفحه                           | تصویر            |
| `[file_ocr]`     | انتخاب فایل تصویر/PDF/TIFF ((بطور پیش‌فرض، متن استخراج می‌کند)| تصویر، PDF، TIFF |
| `[file_read]`    | انتخاب سند متنی                             | TXT، کد، PDF   |
| `[file_audio]`   | انتخاب فایل صوتی                                | MP3، WAV، OGG    |

### پرامپت‌های سفارشی نمونه

- **OCRِ سریع:** `OCRِ من:[file_ocr]`
- **ترجمه‌ی تصویر:** `ترجمه تصویر:متن را از این تصویر استخراج و به فارسی ترجمه کن. [file_ocr]`
- **تحلیل فایل صوتی:** `خلاصه صوت:به این فایل ضبط‌شده گوش کن و نکات اصلی را خلاصه کن. [file_audio]`
- **عیب‌یاب کد:** `عیب‌یاب:اشکالات این کد را پیدا کن و توضیحشان بده: [selection]`

**توجه:** برای دسترسی به همه‌ی امکانات هوش مصنوعی، به یک اینترنت فعال نیاز دارید. فایل‌های TIFF چندصفحه‌ای بطور خودکار پردازش می‌شوند.

## تغییرات برای نسخه‌ی ۳.۶.۰
*   **سیستمِ راهنما**: یک فرمانِ راهنما (`H`) در لایه‌ی فرمان‌ها افزوده شد تا فهرستی با دسترسیِ آسان از تمامِ میانبرها و عملکردهای آن‌ها ارائه دهد.
*   **تحلیلِ ویدیویِ آنلاین:** پشتیبانی گسترش یافت تا ویدیوهای **توییتر (X)** را نیز شامل شود. همچنین، تشخیصِ URL و پایداری بهبود یافت تا تجربه‌ای مطمئن‌تر فراهم شود.
*   **مشارکت در پروژه:** یک کادر گفتگوی اهدای اختیاری برای کاربرانی که مایل به حمایت از به‌روزرسانی‌های آینده و رشدِ مستمرِ پروژه هستند، اضافه شد.

## تغییرات برای نسخه‌ی ۳.۵.۰
*   **لایه‌ی فرمان‌ها** یک سیستم لایه‌ی فرمان‌ها معرفی شد (پیش‌فرض: `NVDA+Shift+V`) برای گروهبندی میانبرها تحت یک کلید اصلی. به عنوان مثال، به جای اینکه برای ترجمه کردن `NVDA+Control+Shift+T` را بفشارید، حالا `NVDA+Shift+V` را فشرده، سپس `T` را می‌زنید.
*   **تحلیل آنلاین ویدئو:** یک ویژگی جدید برای تحلیل مستقیم ویدئوهای یوتیوب و اینستاگرام با ارائه‌ی URL افزوده شد.

## تغییرات برای نسخه‌ی ۳.۱.۰
*   **حالتِ خروجیِ مستقیم:** یک گزینه برای رد کردن پنجره‌ی گفتگو افزوده شده که بتوانید پاسخ‌های هوش مصنوعی را مستقیماً به صورت صوتی بشنوید تا تجربه‌ای سریع‌تر و روان‌تر داشته باشید.
*   **ادغام در کلیپ‌بورد:** تنظیم جدیدی برای کپی کردن خودکار پاسخ‌های هوش مصنوعی در کلیپ‌بورد اضافه شد.

## تغییرات برای نسخه‌ی ۳.۰
*   **زبان‌های جدید:** ترجمه‌های فارسی و ویتنامی افزوده شدند.
*   **مدل‌های بیشتری از هوش مصنوعی:** لیست انتخاب مدل با پیشوندهای واضح ([رایگان]، [حرفه‌ای]، [خودکار]) بازساماندهی شد تا کاربران بتوانند به راحتی مدل‌های رایگان و مدل‌های پولی با محدودیت نرخ را از هم تشخیص دهند. پشتیبانی از Gemini 3.0 Pro و Gemini 2.0 Flash Lite نیز اضافه شد.
*   **پایداری در تایپ صوتی:** پایداری قابلیت تایپ صوتی هوشمند به طور قابل توجهی بهبود یافته است. همچنین یک بررسی ایمنی اضافه شده که کلیپ‌های صوتی کوتاه‌تر از ۱ ثانیه را نادیده می‌گیرد تا از بُروز پاسخ‌های غیرواقعی هوش مصنوعی و خطاهای خالی جلوگیری شود.
*   **مدیریت فایل‌ها:** مشکلی برطرف شد که باعث می‌شد بارگذاری فایل‌هایی با نام‌های غیر انگلیسی دچار خطا شود.
*   **بهینه‌سازی پرامپت‌ها:** منطق ترجمه بهبود یافته و نتایج بینایی به صورت ساختاریافته ارائه شده‌اند.

## تغییرات برای نسخه‌ی ۲.۹
*   **ترجمه‌ی فرانسوی افزوده شد.**
*   **تنظیمات Markdown:** گزینه‌ی جدیدی در تنظیمات به عنوان «پاک کردن Markdown در گفتگو» افزوده شد. غیرفعال کردن این گزینه به کاربران اجازه می‌دهد تا نحوه‌ی نوشتار خام Markdown را (مثلا، `**`, `#`) در پنجره‌ی گفتگو مشاهده کنند.
*   **مدیریت پنجره‌های محاوره‌ای:** مشکلی برطرف شد که باعث می‌شد پنجره‌های «ویرایش متن» یا گفتگو چندین بار باز شوند یا فکوس به درستی روی آن‌ها قرار نگیرد.
*   **بهبودی‌های تجربه‌ی کاربری:** عنوان پنجره‌های انتخاب فایل به صورت استاندارد به «باز کردن» تغییر یافت و اعلان‌های صوتی تکراری (مانند «در حال باز کردن منو...») حذف شد تا تجربه‌ی روان‌تری فراهم شود.

## تغییرات برای نسخه‌ی ۲.۸
* ترجمه‌ی ایتالیایی افزوده شد.
* **اعلام وضعیت:** فرمان جدید (NVDA+Control+Shift+I) برای اعلام وضعیت فعلی افزونه (مثلا «در حال آپلود...» یا «در حال تحلیل...» اضافه شد.
* **خروجی HTML:** دکمه‌ی «ذخیره کردن محتوا» در پنجره‌های نتیجه اکنون خروجی را به صورت یک فایل HTML قالب‌بندی‌شده ذخیره می‌کند و سبْک‌هایی مانند عنوان‌ها و متن پررنگ را حفظ می‌کند.
* **رابط کاربری تنظیمات:** چیدمان پنل تنظیمات با گروه‌بندی دسترس‌پذیر بهبود یافته است.
* **مدل‌های جدید:** پشتیبانی از Gemini-flash-latest و Gemini-flash-lite-latest افزوده شد.
* **زبان‌ها:** نپالی به زبان‌های پشتیبانی‌شده افزوده شد.
* **منطق منوی پالایش:** یک مشکل اساسی برطرف شد؛ اینکه اگر زبان رابط کاربری NVDA انگلیسی نباشد، فرمان‌های پالایش متن دچار خطا می‌شدند.
* **تایپ صوتی:** شناسایی سکوت بهبود یافته تا از تولید خروجی متنی نادرست هنگام نبود ورودی گفتار جلوگیری شود.
* **تنظیمات بروزرسانی:** گزینه‌ی «بررسی بروزرسانی در آغاز» حالا به طور پیش‌فرض غیرفعال است تا با سیاست‌های فروشگاه افزونه‌ها مطابقت داشته باشد.
* کد پاک‌سازی شد.

## تغییرات برای نسخه‌ی ۲.۷
* ساختار پروژه به قالب رسمی افزونه‌های NV Access منتقل شد تا مطابقت بیشتری با استانداردها داشته باشد.
* منطق تلاش مجدد خودکار برای خطاهای HTTP 429 (درخواست بیش از حد مجاز) پیاده‌سازی شد تا در زمان ترافیک بالا، اطمینان از عمل‌کرد تضمین شود.
* دستورهای ترجمه برای دقت بالاتر و مدیریت بهتر منطق «جابجایی هوشمند» بهینه‌سازی شدند.
* ترجمه‌ی روسی بروز شد.

## تغییرات برای نسخه‌ی ۲.۶
* ترجمه‌ی روسی افزوده شد. (با سپاس از nvda-ru).
* پیام‌های خطا به‌روزرسانی شدند تا بازخورد توصیفی‌تر و واضح‌تری درباره اتصال ارائه دهند.
* زبان مقصد بطور پیش‌فرض به انگلیسی تغییر یافت.

## تغییرات برای نسخه‌ی ۲.۵
* فرمان بومی OCR فایل (NVDA+Control+Shift+F) افزوده شد.
* دکمه‌ی «ذخیره کردن گفتگو» به پنجره‌های محاوره‌ای نتایج افزوده شد.
* پشتیبانی کامل از بومی‌سازی (i18n) پیاده‌سازی شد.
* بازخورد صوتی به ماژول بومی «tones» NVDA منتقل شد.
* استفاده از Gemini File API جایگزین شد تا مدیریت بهتری برای فایل‌های PDF و صوتی فراهم شود.
* مشکل توقف افزونه هنگام ترجمه‌ی متنی که شامل آکولاد بود، رفع شد.

## تغییرات برای نسخه‌ی ۲.۱.۱
* مشکلی که در آن متغیر [file_ocr] به درستی در پرامپت‌های سفارشی کار نمی‌کرد برطرف شد.

## تغییرات برای نسخه‌ی ۲.۱
* همه‌ی کلیدهای میانبر افزونه به استفاده از ترکیب NVDA+Control+Shift استاندارد شدند تا با چیدمان لپ‌تاپ NVDA و کلیدهای میانبر سیستم تداخل نداشته باشند.

## تغییرات برای نسخه‌ی ۲.۰

* حافظه‌ی مکالمه اضافه شد تا نتایج در گفتگوها بر اساس زمینه‌ی گفتگو دقیق‌تر شوند.

## تغییرات برای نسخه‌ی ۱.۵
